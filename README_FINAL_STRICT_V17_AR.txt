PriceBot FINAL STRICT V17
=========================

هذه النسخة مبنية من PriceBot_FAST_SAFE_NO_TIMEOUT.zip مع تنظيف مسار القرار النهائي.

أهم التغييرات:
1) حذف fast_matcher من الحزمة لأنه لم يعد جزءاً من مسار الإنتاج.
2) كل النصوص والصور تمر عبر matcher_v4 كمحرك قرار نهائي.
3) إلغاء fallback القديم بعد الصور؛ الصورة الضعيفة لا تتحول إلى بحث نصي عشوائي.
4) منع عرض السعر داخل قوائم التوضيح قبل اختيار المنتج/الشكل/الجرعة بالضبط.
5) منع بدائل الأدوية التلقائية. البدائل الدوائية تحتاج صيدلي أو substitution_group_id لاحقاً.
6) بدائل الكوزمتك لا تظهر إلا من قرار V4 strict وبنفس النوع والاستخدام.
7) التوفر الفارغ/غير المعروف لم يعد يعتبر متوفراً، ولا يسمح بالحجز.
8) تحسين حالة المحادثة حتى لا تضيع الخيارات أو يختلط آخر منتج مع قائمة توضيح.
9) تشغيل warmup لفهارس V4 بالخلفية بعد بدء السيرفر.
10) رفض queue full بكود 503 بدل 200 حتى لا تضيع الرسائل بصمت.
11) تعطيل دخول الأدمن بالمفتاح داخل URL؛ الدخول من /admin/login فقط.

اختبارات التحقق:
- pytest -q
- python acceptance_tests_v3.py
- python acceptance_tests_v4.py
- python acceptance_tests_final_v17.py

مهم:
هذه النسخة تقلل الأخطاء الكبيرة وتمنع التخمين، لكنها لا تلغي الحاجة إلى كتالوج نظيف. دقة البوت تعتمد على أعمدة brand/category/form/strength/size/aliases/ocr_keywords/available.
