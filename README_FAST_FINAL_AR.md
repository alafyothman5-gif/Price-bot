# PriceBot Fast Final Image/Text Match

هذه النسخة تضيف:

- رد فوري عند استلام الصورة: "جاري فحص الصورة".
- تصغير الصورة قبل إرسالها إلى OpenRouter لتقليل الوقت والتكلفة.
- Prompt قصير للصور: AI يقرأ النص فقط ولا يختار السعر.
- Timeout للصورة حتى لا يعلق البوت دقائق طويلة.
- مقارنة المنتج محلياً مع قاعدة بيانات الصيدلية فقط.
- إذا المنتج غير متوفر وكان كوزمتك: يعرض بدائل متوفرة مباشرة ويرشح أفضل بديل.
- لا تحذف المنتجات ولا قاعدة البيانات ولا توكن واتساب ولا مفتاح OpenRouter الموجود.

بعد الرفع إلى GitHub شغل DEPLOY_FROM_GITHUB.sh بالطريقة المعتادة.
