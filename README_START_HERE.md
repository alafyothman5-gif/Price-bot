# PriceBot — النسخة النهائية لصيدلية بدر البشرية

هذه نسخة GitHub جاهزة للتشغيل على السيرفر الحالي مع الحفاظ على `.env` وملفات المنتجات والطلبات الموجودة على السيرفر.

## أهم ما تم إضافته

- لوحة إدارة أكثر احترافية ومناسبة للهاتف.
- لوحة طلبات منظمة مع إحصائيات، بحث، فلترة، فتح واتساب الزبون، وفتح صورة الطلب/الروشتة عند وجودها.
- خانة في صفحة الإعدادات لرقم أدمن واتساب الذي يستقبل تنبيهات الطلبات والصور.
- عند تأكيد الحجز:
  - الطلب ينزل في لوحة الطلبات.
  - الزبون يستلم رسالة تأكيد فقط.
  - الأدمن يستلم رسالة واتساب خاصة إذا كان رقم الأدمن مضبوطاً.
- منع إرسال رسالة “طلب جديد من PriceBot” إلى الزبون.
- قراءة صور المنتجات قدر الإمكان عبر Gemini Vision عند توفر مفتاح Gemini.
- إذا كانت الصورة روشتة أو غير واضحة أو لم يتم التعرف عليها بثقة، يتم تحويلها للأدمن ولا يعطي البوت أي نصيحة طبية.
- منع النصائح الطبية بالكامل: لا وصفات، لا جرعات، لا ترشيح علاج للأعراض.
- البحث عن المنتجات بالنص، وبالأسماء البديلة، وبالخطأ الإملائي البسيط.

## إعداد رقم الأدمن

افتح:

```text
/admin/settings?key=PriceBotAdmin2026
```

ثم ضع رقم الأدمن في خانة:

```text
رقم أدمن واتساب للتنبيهات
```

الصيغة المطلوبة:

```text
2189XXXXXXXX
```

## الصور والروشتات

- صورة علبة منتج واضحة: يحاول البوت استخراج اسم المنتج ومطابقته مع المنتجات.
- إذا وجد المنتج: يرد بالسعر والتوفر.
- إذا لم يجد مطابقة: يحولها للأدمن.
- روشتة/وصفة/صورة غير واضحة: تتحول للأدمن مباشرة.

ملاحظة: قراءة الصور تعتمد على وجود مفتاح Gemini صالح ومفعل من صفحة الإعدادات.

## الذكاء الاصطناعي المجاني

الترتيب الافتراضي:

```text
gemini,openrouter,groq
```

الموديلات المقترحة:

```text
Gemini: gemini-2.5-flash-lite
OpenRouter: openrouter/free
Groq: llama-3.1-8b-instant
```

## النشر على السيرفر

بعد رفع الملفات إلى GitHub، شغل من السيرفر:

```bash
cd /opt/pricebot && bash DEPLOY_ON_SERVER.sh
```

أو استخدم سكربت النشر الموجود لديك إذا كنت تستخدمه مع GitHub.

## روابط مهمة

لوحة المنتجات:

```text
/admin?key=PriceBotAdmin2026
```

لوحة الطلبات:

```text
/admin/orders?key=PriceBotAdmin2026
```

رفع ملف المنتجات:

```text
/admin/upload?key=PriceBotAdmin2026
```

الإعدادات:

