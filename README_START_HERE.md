# PriceBot — نسخة صيدلية بدر البشرية مع AI API حقيقي

هذه نسخة جاهزة لـ GitHub والتشغيل على السيرفر.

## ما الجديد

- ردود مخصصة باسم: صيدلية بدر البشرية.
- المدينة: أجدابيا.
- ساعات العمل: 24 ساعة.
- التوصيل غير متوفر حالياً.
- لوحة منتجات مناسبة للهاتف.
- لوحة طلبات مناسبة للهاتف.
- رفع CSV / Excel.
- دعم أسماء بديلة للمنتجات.
- ذكاء محلي مجاني للبحث الأساسي.
- دعم ذكاء اصطناعي حقيقي عبر API اختياري:
  - OpenRouter
  - OpenAI
  - Gemini
  - Custom Chat Completions

## التشغيل على السيرفر

بعد رفع الملفات إلى GitHub، استخدم سكربت النشر الموجود لديك أو شغل:

```bash
cd /opt/pricebot && bash DEPLOY_ON_SERVER.sh
```

السكريبت يحافظ على:

- `.env`
- `products.csv`
- `orders.csv`

ولا يضع أي أسرار داخل GitHub.

## تفعيل الذكاء الاصطناعي الحقيقي

بعد نشر النسخة، افتح:

```text
https://46.101.148.246.sslip.io/admin/settings?key=PriceBotAdmin2026
```

ثم:

1. فعّل خيار: تفعيل AI API.
2. اختر المزود:
   - OpenRouter غالباً الأسهل لأنه يقبل موديلات متعددة.
   - OpenAI إذا عندك مفتاح OpenAI.
   - Gemini إذا عندك مفتاح Google Gemini.
3. ضع الموديل.
4. الصق مفتاح API.
5. اضغط حفظ.
6. اضغط اختبار AI.

لا ترسل مفتاح API في الدردشة ولا ترفعه إلى GitHub.

## كيف يعمل AI داخل البوت

البوت لا يفتح دردشة عشوائية. AI يستخدم فقط لفهم نية الزبون مثل:

- كم سعر البروفين؟
- عندكم بندول؟
- نبي شي للصداع
- في أوجمنتين؟

ثم يبحث في قائمة المنتجات الموجودة فقط. لا يخترع منتجاً غير موجود.

لو الزبون سأل عن جرعة أو استعمال أو حالة طبية، البوت لا يعطي نصيحة علاجية، بل يطلب مراجعة الصيدلي.
