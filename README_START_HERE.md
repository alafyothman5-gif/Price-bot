# PriceBot — WhatsApp Product & Orders Bot

هذه نسخة مرتبة جاهزة للرفع إلى GitHub ثم التشغيل على السيرفر.

## ماذا تحتوي النسخة؟

- `app.py` كامل ونظيف.
- لوحة منتجات محسنة للموبايل.
- لوحة طلبات محسنة للموبايل.
- رفع CSV/Excel للمنتجات.
- إدخال منتجات بالجملة.
- بحث عربي/إنجليزي مع أسماء بديلة.
- تسجيل حجز الطلبات.
- تصدير الطلبات CSV.
- Webhook واتساب كما هو.
- لا يحتوي على توكنات أو أسرار.

## ملفات لا ترفعها إلى GitHub

لا ترفع:

- `.env`
- `products.csv` الحقيقي إذا فيه بيانات عميل خاصة
- `orders.csv`
- `venv`
- أي ملفات backup

## التشغيل على السيرفر الحالي

بعد رفع الملفات إلى GitHub، ادخل إلى السيرفر وشغّل ملف:

```bash
bash DEPLOY_ON_SERVER.sh
```

السكريبت يحافظ على:

- `.env`
- `products.csv`
- `orders.csv`

ويأخذ نسخة احتياطية قبل الاستبدال.

## روابط مهمة بعد التشغيل

لوحة المنتجات:

```text
/admin?key=PriceBotAdmin2026
```

لوحة الطلبات:

```text
/admin/orders?key=PriceBotAdmin2026
```

الصحة:

```text
/health
```
