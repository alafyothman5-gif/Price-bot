# PriceBot Final Variant/Alternatives Edition

هذه النسخة تضيف:

- التفريق بين الأصناف المتشابهة مثل بنادول أدفانس / بنادول إكسترا.
- عند وجود أكثر من شكل أو تركيز لنفس الاسم، يسأل الزبون يختار الرقم أو النوع.
- دعم أعمدة إضافية للمنتجات: المادة الفعالة، الشركة/المنشأ، الشكل، التركيز، العبوة، رابط الصورة.
- عرض بدائل بنفس المادة/التركيز إذا كانت البيانات موجودة في ملف المنتجات، بدون إعطاء جرعات أو نصائح طبية.
- منع إعطاء الجرعات أو العلاج. البوت يعرض سعر/توفر/حجز فقط.
- بقاء تنبيهات الأدمن والصور والروشتات ولوحة الإدارة من النسخة السابقة.

## أعمدة المنتجات المفضلة

```csv
name,aliases,active_ingredient,company,form,strength,pack,price,available,notes,image
```

مثال:

```csv
بنادول إكسترا,بنادول اكسترا|بندول احمر|panadol extra,Paracetamol+Caffeine,GSK,أقراص,500mg/65mg,شريط,8 د.ل,متوفر,,https://example.com/panadol-extra.jpg
أموكلان 625,اموكلان|amoclan|amoxicillin clavulanate,Amoxicillin+Clavulanate,أردني,أقراص,625mg,علبة,18 د.ل,متوفر,,
```

## التشغيل على السيرفر

بعد رفع الملفات إلى GitHub:

```bash
cd /opt/pricebot && bash DEPLOY_ON_SERVER.sh
```

ثم افتح لوحة الإعدادات وضع رقم أدمن واتساب.


## ذاكرة البوت وتقليل استهلاك AI

أضيفت صفحة ذاكرة من لوحة الأدمن:

```text
/admin/memory?key=PriceBotAdmin2026
```

الذاكرة تحفظ:
- العبارات التي فهمها AI سابقًا حتى لا يستدعي API مرة أخرى لنفس العبارة.
- نتائج صور المنتجات المتكررة عبر hash الصورة.
- تصحيحات الأدمن مثل: `بندل -> بنادول`.
- عداد استخدام AI اليومي حسب المزود.

الملف المحفوظ هو `memory.json` داخل مجلد المشروع، ويتم حفظه أثناء النشر مع المنتجات والطلبات.
