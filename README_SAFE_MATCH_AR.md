# PriceBot SAFE IMAGE/TEXT MATCH V1

هذه النسخة تضيف طبقة أمان لمطابقة الصور والنصوص:

- AI يقرأ الصورة/النص فقط ولا يقرر السعر.
- قاعدة البيانات هي المصدر الوحيد للسعر والتوفر.
- منع اختيار منتج من نوع مختلف: غسول ≠ مرطب، أقراص ≠ شراب، حقن ≠ كبسولات.
- إذا المنتج واضح لكنه غير موجود، يرد للزبون: غير متوفر.
- إذا الصورة غير واضحة، يطلب صورة أوضح أو كتابة الاسم.
- إذا المنتج كوزمتك وغير متوفر، يعرض على الزبون خيار البدائل.
- البدائل للكوزمتك فقط، ولا يقترح بدائل علاجية للأدوية.

بعد رفع هذه النسخة إلى GitHub، شغّل أمر النشر المعتاد من السيرفر.
